# house-price-prediction
 California House Price Prediction Web App

This project is a **Machine Learning web application** that predicts house prices in California based on various housing features.  
The model is built using **Random Forest Regression** and deployed using **Streamlit Cloud**
Live Web App
You can access the deployed app here:
👉 https://your-streamlit-app-link.streamlit.app

Project Overview:-
The goal of this project is to build a machine learning model that can predict house prices using features such as:

- Longitude
- Latitude
- Housing Median Age
- Total Rooms
- Total Bedrooms
- Population
- Households
- Median Income
- Ocean Proximity

The project includes:

- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
- Web application development
- Cloud deployment
Machine Learning Model:-
Algorithm used:-

1)Random Forest Regressor

Why Random Forest?

- Handles nonlinear relationships well
- Reduces overfitting
- Works well for tabular datasets
